# .files

These are my dotfiles, forked from Lars Kappert's [dotfiles](https://github.com/webpro/dotfiles). Take anything you want, but at your own risk.

I've stripped this down a lot to minimize administrative overhead.

## Installation

On a sparkling fresh installation of macOS:

```bash
sudo softwareupdate -i -a
xcode-select --install
```

1. Clone the repo.

```bash
git clone https://github.com/andrewbaughman/dotfiles.git
```

2. Use the install-* files to install things. Use the set-* files to set things. I made a shortcut to do them all.

```bash
source install.sh
```
